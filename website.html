Setup Website Booking Widget
The website widget booking engine integration required you to copy paste the following code to your website main html page. 

  
    <form method="GET" action="">
      <input type="submit" value="Book Room" />
    </form>
Optionally you can display extra field to show check in date ∓ check out date. 
  
    <form method="GET" action="">
      <label for="checkin" />
      <input type="date" name="checkin" />
      <label for="checkout" />
      <input type="date" name="checkout" />
      <input type="submit" value="Book Room" />
    </form>
See booking in action

Open your website and the booking button should be shown and ready to receive booking.
